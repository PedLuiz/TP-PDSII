Descrição detalhada

Classe responsável pelo gerenciamento de jogadores e suas informações no sistema.

A classe Sistema permite o cadastro, remoção, e listagem de jogadores. Além disso, ela gerencia a persistência de dados em um arquivo e a exibição de informações sobre os jogadores cadastrados.
